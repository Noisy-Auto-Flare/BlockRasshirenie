# YouTube Limiter — Расширение для ограничения просмотра YouTube

[![Chrome Web Store](https://img.shields.io/badge/Chrome%20Web-Store-4285F4.svg)](https://chrome.google.com/webstore)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## 📖 Описание

**YouTube Limiter** — умное расширение для браузера Chrome, которое помогает контролировать время просмотра YouTube. Распознаёт активные вкладки с видео и автоматически блокирует доступ после достижения лимита времени.

---

## ✨ Основные возможности

### 🎯 Умный таймер
- **Распознаёт только вкладки с видео** — обычные страницы YouTube (поиск, каналы) не учитываются
- **Динамическая генерация иконки** — отображает оставшееся время прямо на иконке расширения

### 🔒 Надёжная защита от обхода
- **Блокировка через Service Worker** — перехват всех запросов к YouTube
- **Защита на уровне DNS** — блокировка через `declarativeNetRequest` API
- **Правила загружаются с GitHub** — автоматическое обновление списка заблокированных доменов

### ⚙️ Настройки с защитой
- **Время просмотра** — установите лимит времени (по умолчанию 30 минут)
- **Кулдаун** — время блокировки после истечения лимита (по умолчанию 60 минут)
- **Защита редактирования** — после сохранения настроек нельзя изменить их в течение 5 минут

### 📋 Панель открытых вкладок
- **Список всех YouTube вкладок** с иконками по типу страницы:
  - 🎬 — страница с видео (активный просмотр)
  - 🔍 — страница поиска
  - 📺 — страница канала
  - ▶️ — другие страницы YouTube
- **Быстрое переключение между вкладками** одним кликом
- **Закрытие ненужных вкладок** через кнопку крестика

### ⏱️ Красивая страница блокировки
- Анимированный обратный отсчёт до разблокировки
- Прогресс-бар с визуализацией оставшегося времени
- Автоматическое переключение стиля при приближении разблокировки (⚠️ warning → 🚨 danger)

## 🔧 Установка

### Вариант 1: Загрузка из репозитория

1. **Скачайте репозиторий:**
   ```bash
   git clone https://github.com/Noisy-Auto-Flare/BlockRasshirenie.git
   cd BlockRasshirenie
   ```

2. **Откройте страницу расширений Chrome:**
   - Перейдите по адресу `chrome://extensions/`
   - Или нажмите `Menu → Дополнительные возможности → Расширения`

3. **Включите режим разработчика:**
   - Найдите переключатель "Режим разработчика" в правом верхнем углу
   - Включите его

4. **Загрузите расширение:**
   - Нажмите кнопку "Загрузить распакованное"
   - Выберите папку `BlockRasshirenie`
   - Расширение установлено! 🎉

## 📱 Использование

### Настройка расширения

1. **Откройте страницу настроек:**
   - Перейдите по адресу `chrome://extensions/`
   - Найдите YouTube Limiter в списке расширений
   - Нажмите на ссылку "Открыть страницу настроек" (если есть)
   - Или кликните правой кнопкой мыши по иконке расширения → "Настройки"

2. **Установите параметры:**
   ```
   Время просмотра: [число] минут  — лимит активного времени за сессию
   Кулдаун:          [число] минут  — время блокировки после истечения лимита
   ```

3. **Сохраните настройки** — нажмите кнопку "💾 Сохранить"

4. **Защита от редактирования:** После сохранения нельзя изменить настройки в течение 5 минут (показано на странице)

### Просмотр открытых вкладок

1. **Откройте панель вкладок:**
   - Кликните по иконке расширения в панели инструментов Chrome
   - Откроется popup со списком всех YouTube вкладок

2. **Навигация:**
   - ⬅️ — перейти на вкладку (закрыть popup)
   - ✕ — закрыть вкладку

3. **Статус бар показывает:**
   - 🟢 Зелёный — время просмотра (оставшиеся минуты:секунды)
   - 🟠 Жёлтый — кулдаун активен

### Компоненты расширения

#### 📄 manifest.json
Манифест v3 с разрешениями:
- `tabs` — управление вкладками и перехват навигации
- `storage` — сохранение настроек и состояния таймера
- `declarativeNetRequest` — блокировка запросов на уровне браузера

#### 🔧 background.js (Service Worker)
Класс `YouTubeLimiter`:
```javascript
class YouTubeLimiter {
  init()                    // Инициализация и загрузка настроек
  handleNewTab(tab)        // Обработка создания/обновления вкладки
  decreaseTime()           // Уменьшение времени на 1 секунду
  checkCooldown()          // Проверка статуса кулдауна
  updateTimer()            // Обновление иконки с таймером
  generateIcon(text, bg, fg) // Генерация иконки через Canvas
}
```

#### 🌐 content.js (Content Script)
Класс `YouTubeContentManager`:
- Обработка сообщений от Service Worker
- Периодическая проверка статуса блокировки
- Перенаправление на blocked.html при блокировке

#### ⚙️ options.html + options.js
Страница настроек с классом `OptionsPage`:
```javascript
class OptionsPage {
  init()                   // Загрузка DOM и настроек
  loadSettings()          // Загрузка из chrome.storage.local
  saveSettings()          // Сохранение с защитой (lockTime)
  checkLockStatus()       // Проверка блокировки редактирования
}
```

#### 📋 popup.html + popup.js
Класс `TabsManager`:
```javascript
class TabsManager {
  fetchTabs()             // Получение всех YouTube вкладок
  renderTabs()            // Рендеринг списка с иконками по типу
  navigateAndClosePopup(id) // Переход на вкладку + закрытие popup
  closeTabAndRemove(id)   // Закрытие вкладки + обновление списка
}
```

#### 🚫 blocked.html + blocked.js
Класс `BlockedPage`:
- Анимированный обратный отсчёт до разблокировки
- Прогресс-бар с визуализацией
- Автоматическое переключение на YouTube после истечения кулдауна

## 🔐 Механизм блокировки

### Уровень 1: Service Worker (background.js)
```javascript
chrome.tabs.onCreated.addListener(handleNewTab);
chrome.tabs.onUpdated.addListener((id, info, tab) => handleNewTab(tab));
```
При создании/обновлении вкладки проверяется кулдаун и перенаправляется на `blocked.html`

### Уровень 2: Content Script (content.js)
Периодическая проверка кулдауна каждые 5 секунд:
```javascript
setInterval(() => checkBlockStatus(), 5000);
```

### Уровень 3: Declarative Net Request
Загрузка правил с GitHub и применение через `chrome.declarativeNetRequest.updateSessionRules()` — блокировка на уровне DNS

### Запуск локально

1. Откройте `chrome://extensions/`
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное"
4. Выберите папку проекта

### Отладка

**Background Service Worker:**
```
chrome://service-worker-management/
```

**Console логов:**
- Background: `chrome://extensions → YouTube Limiter → Service Worker → Inspect`
- Content script: откройте DevTools на странице YouTube (F12)

### Добавление новых правил блокировки

Правила хранятся в `rules.json`:
```json
[
  "www.youtube.com",
  "youtube.com",
  "m.youtube.com"
]
```

Или загрузка с GitHub:
```javascript
const rulesUrl = 'https://raw.githubusercontent.com/Noisy-Auto-Flare/BlockRasshirenie/master/rules.json';
```
